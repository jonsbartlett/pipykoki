pykoki is a ctypes-based binding for libkoki

Requirements
------------

 * [libkoki](https://www.studentrobotics.org/trac/wiki/libkoki)
 * [v4l2](http://pypi.python.org/pypi/v4l2)

Installation
------------

pykoki can be installed used python distutils like this:

~~~~~~~~~~~~
% ./setup.py install
~~~~~~~~~~~~

That's it!
